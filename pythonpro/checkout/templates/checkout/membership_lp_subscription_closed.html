Descrição

No curso de Django, você vai aprender a utilizar o framework mais utilizado da linguagem Python, o Django!

O Django é um framework que vai te ajudar a construir sistemas web de uma forma simples, rápida, segura e escalável.

O Django é um dos pioneiros do conceito "Don't Repeat Yourself", que, em português significa "Não se repita". Você vai construir sistemas poderosos com poucas linhas de código, pois o Django se responsabiliza para você não precisar repetir código!
