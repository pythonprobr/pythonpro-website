PyTools

No curso PyTools você vai aprender como equipes de programadores trabalham e colaboram entre si. Assim, você vai poder finalmente ter a capacidade de ser um deles!

Você vai aprender a trabalhar com a ferramenta que todo mundo usa para versionar código e contribuir com projetos de desenvolvimento de software: o Git.

Django

No curso de Django, você vai aprender a utilizar o framework mais utilizado da linguagem Python, o Django!

O Django é um framework que vai te ajudar a construir sistemas web de uma forma simples, rápida, segura e escalável.

O Django é um dos pioneiros do conceito "Don't Repeat Yourself", que, em português significa "Não se repita". Você vai construir sistemas poderosos com poucas linhas, pois o Django se responsabiliza para você não precisar repetir código!
